# Stochastic model of curvature controlled tissue growth incorporating mechanical cell interactions

Computer code in Julia used for discrete model simulations presented in the research article "Mechanistic Interpretation of biological tissue growth experiments with a computational model". Preprint available at " ".

The simulation code is programmed in Julia and organised into the “src” folder. A separate folder is dedicated to the collective plotting code. A folder named “Scripts” contains the code responsible for generating all paper figures. It is important to note that these images were subsequently exported to Inkscape for post-processing and figure labeling.
